Description:

Schedule Builder is a scheduling application with simple functionality.
A user can make a task list and put checks near the complete tasks.
The purpose of this application is to help its users develop self-discipline
by reflecting on their daily accomplishments and/or possible failure
to complete certain planned tasks.
